# Blog Editor

Desktop blog editor built with React + Vite + Electron.

## Features

- Create, edit, and delete Markdown blog posts
- Upload images to blog assets
- Git status and sync workflow
- Local blog root and repository settings

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Package (Portable, Windows)

```bash
npm run dist:portable
```
